# DevCamper API

> Backend API for DevCamper application, which is bootcamp directory website

## Usage

Rename ".env.example" to ".env" and update values/settings to your own

## Install dependencies

```
npm install
```

## Build app

```
npm run build
```

## Run app

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```

## Database

```
# Seed database
npm run db:seed

# Drop database
npm run db:drop
```

## Documentation

```
http://localhost:5000/
```
